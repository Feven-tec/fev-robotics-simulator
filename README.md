 # Fev Robotics AI Simulator

A web-based AI robotics simulator demonstrating a virtual robot that responds to commands, simulates movements, and acts as an AI assistant. Showcases coding skills, AI logic, and robotics vision.

## Features
- Text-based AI chatbot responding to user inputs
- Virtual robot messages and interactions
- Simple interactive UI

## Tech Stack
- React.js
- JavaScript
